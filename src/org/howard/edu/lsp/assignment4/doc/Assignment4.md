CRC Cards: ATC System

CRC Card
Class: Ground Receiver
Responsibilities:
Receive aircraft data packets transmitted by transponders
Validate packet reception before processing
Forward received packets to the decoding component
Collaborators (if any):
Transponder
Packet Decoder
Assumptions (if any):
Aircraft within range continuously broadcast transponder packets
CRC Card
Class: Packet Decoder
Responsibilities:
Decode high-density aircraft data packets
Extract aircraft identity and flight information
Create or update aircraft records using decoded data
Collaborators (if any):
Ground Receiver
Aircraft
Aircraft Registry
Assumptions (if any):
N/A
CRC Card
Class: Aircraft
Responsibilities:
Maintain identifying information for an aircraft (flight number, aircraft type)
Store current flight state such as altitude, speed, heading, and location
Update stored flight data when new decoded information is received
Collaborators (if any):
Packet Decoder
Aircraft Registry
Assumptions (if any):
Each aircraft is uniquely identifiable by its flight number
CRC Card
Class: Aircraft Registry
Responsibilities:
Maintain the collection of aircraft currently tracked by the system
Update aircraft records when new flight data arrives
Provide aircraft data to system components requesting it
Collaborators (if any):
Aircraft
PacketDecoder
DisplayService
SafetyMonitor
ControllerQuery
Assumptions (if any):
Only aircraft currently being tracked by ATC are stored in the registry
CRC Card
Class: Display Service
Responsibilities:
Retrieve aircraft tracking information from the registry
Prepare aircraft data for graphical presentation
Trigger updates of the controller display at regular intervals
Collaborators (if any):
Aircraft Registry
Controller Display
Assumptions (if any):
N/A
CRC Card
Class: Safety Monitor
Responsibilities:
Analyze aircraft flight data for potential conflicts
Detect dangerous proximity or unsafe flight conditions
Generate alerts when safety thresholds are violated
Collaborators (if any):
Aircraft Registry
Aircraft
Controller Display
Assumptions (if any):
N/A
CRC Card
Class: Controller Display
Responsibilities:
Render aircraft positions and identifiers on the controller screen
Display alerts generated by safety monitoring
Allow controllers to request additional aircraft information
Collaborators (if any):
Display Service
Safety Monitor
Controller Query
Aircraft Registry
Assumptions (if any):
N/A
CRC Card
Class: Controller Query
Responsibilities:
Process controller requests for aircraft details
Retrieve requested information from the aircraft registry
Return detailed aircraft data to the display interface
Collaborators (if any):
Aircraft Registry
Aircraft
Controller Display
Assumptions (if any):
N/A
